Media Cloud for Research

Methodology

Media Cloud is a joint project created by the MIT Center for Civic Media and the Berkman Klein Center for Internet and Society at Harvard University. The website allows users to search individual media sources or across media sources by grouping into collections. The media collections span over 100 countries and 1.4 billion media stories. Media Cloud crawls approximately 59,000 sources and 611,000 stories every day.
